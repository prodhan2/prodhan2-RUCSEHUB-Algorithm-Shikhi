**RUCSEHUB-Algorithm-Shikhi – Short Description**

*RUCSEHUB-Algorithm-Shikhi* হলো একটি learning-based platform যেখানে অ্যালগরিদম (Algorithm) সহজভাবে শেখানো হয় visualization-এর মাধ্যমে, যাতে শিক্ষার্থীরা step-by-step বুঝতে পারে কিভাবে একটি algorithm কাজ করে।
